DATABASE
  CREATE TABLE words (
    id serial PRIMARY KEY,
    word VARCHAR(11) UNIQUE NOT NULL,
    letter_count SMALLINT NOT NULL,
    meaning TEXT NOT NULL,
    points SMALLINT NOT NULL DEFAULT 0,
    vote_count SMALLINT NOT NULL DEFAULT 0
  )

Socket
 - Yeni bir oda yaratma
   * Bir kişi yeni oda yarat butonuna basar
   * Unique bir oda numarası yaratılır
   * Odayı yaratan kişi host olarak atanır 
   * Oda için room açılır
   * Oda için sorular seçilir

 - Varolan odaya katılma
   * Bir kişi oda idsi girer
   * Odanın varlığı kontrol edilir
   * Rooma o kişi bağlandı diye mesaj gönderilir

 - Oda işlemleri
   * Host hariç herkes hazırım ve hazır değilim mesajları yayınlayabilir
   * Herkes hazır oluğunda host oyunu başlatabilir ve oyun başlıyor mesajı yayınlanır
   * Bütün sorular tek seferde front-ende yollanacak
   * Harf alma, süre, soru çözme gibi işlemler front-endde hallolacak
   * Birisi oyunu bitirince sonuçlarını yayınlayacak
   * Oyunu bitenler lobide oyunu bitenlerin sonuçlarını, oyunu bitirmeyenlerin de bitmedi durumunu görebilecek
   * Front-endde sonuçlara göre sıralama yapılacak 
   * Birisi oyun esnasında düşerse diğerlerini şu kişi düştü mesajı yollanacak ve front-endde sonuç sayfasında gri görünecek
